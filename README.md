# Pantalla Digital de Citas - Nissan

Pantalla de señalización digital para mostrar el turno actual y las próximas citas de servicio en el concesionario Nissan.

## Tecnologías

- **React 18** + TypeScript
- **Vite** (bundler)
- **Tailwind CSS** (estilos)

## Desarrollo local

```bash
npm install
npm run dev
```

## Despliegue en Railway

Este proyecto está configurado para desplegarse automáticamente en Railway.

- **Build command:** `npm run build`
- **Start command:** `npm run start`
- Puerto configurado via variable de entorno `$PORT`
